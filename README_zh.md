# 创意工坊

[English](./README.md) | 中文 

### 如何上传我的资源？

* 想要上传资源到创意工坊，你需要在 **此仓库** 创建一条 Pull Request

* 建议使用 https://smms.app/ 或其他全球可访问的图床，或者是你自己的对象存储（中国用户无法访问github raw）

### 发布资源需要遵守的规则：

* 您发布的资源内不得包含Itemsadder的默认资源（包括但不限于材质、模型、.jar文件）
